# 弱口令/暴力破解扫描器

一款自动化弱口令检测工具，通过内置字典对Web登录接口进行暴力破解测试，检测是否存在弱口令漏洞。

## ✨ 功能特点

- ✅ **自动表单识别** - 解析页面中的登录表单，自动提取用户名和密码参数
- ✅ **智能登录判断** - 通过关键词匹配、URL变化、状态码判断登录是否成功
- ✅ **自动路径探测** - 支持自动探测常见登录页面路径
- ✅ **多线程并发** - 支持多线程高速爆破
- ✅ **速率保护** - 自动检测HTTP 429响应，避免被封IP
- ✅ **内置字典** - 内置24个常见用户名和40个常见密码
- ✅ **自定义字典** - 支持自定义用户名/密码或加载字典文件
- ✅ **结果保存** - 支持将发现的凭据导出到文件
