# XSS漏洞扫描器

一款自动化XSS（跨站脚本攻击）漏洞检测工具，用于发现Web应用中GET参数存在的反射型XSS漏洞。

## ✨ 功能特点

- ✅ **自动参数识别** - 自动提取URL中所有GET参数逐一测试
- ✅ **双模式扫描** - 基础模式和高级绕过模式（大小写混淆、编码绕过等）
- ✅ **智能反射检测** - 支持原始反射、HTML实体编码反射、URL编码反射检测
- ✅ **丰富载荷库** - 内置多种HTML标签、事件触发器、闭合规则载荷
- ✅ **实时进度条** - 使用 tqdm 显示测试进度，清晰直观
- ✅ **结果保存** - 支持将发现的漏洞导出到文件
- ✅ **灵活控制** - 可自定义线程数、选择载荷级别
