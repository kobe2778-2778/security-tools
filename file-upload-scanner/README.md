# 文件上传漏洞扫描器

一款自动化文件上传漏洞检测工具，通过尝试上传各种危险后缀名的文件，检测目标Web应用是否存在文件上传漏洞。

## ✨ 功能特点

- ✅ **自动表单检测** - 自动识别页面中的文件上传表单
- ✅ **多后缀名测试** - 内置50+危险后缀名变体
- ✅ **多Content-Type绕过** - 自动尝试不同Content-Type伪造
- ✅ **智能路径探测** - 自动从响应中提取文件路径，并尝试常见路径
- ✅ **执行验证** - 上传后访问文件，验证是否能被服务器执行
- ✅ **实时进度条** - 使用 tqdm 显示测试进度
- ✅ **结果保存** - 支持将发现的漏洞导出到文件
- ✅ **无害测试** - 上传的文件只输出随机字符串，不会对服务器造成危害
