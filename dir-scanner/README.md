# 目录/文件爆破扫描器

一款多线程Web目录扫描工具，用于发现目标Web服务器上隐藏的目录、文件和敏感路径。

## ✨ 功能特点

- ✅ **多线程并发** - 速度飞快，默认30线程，可自定义
- ✅ **实时进度条** - 使用 tqdm 显示扫描进度，清晰直观
- ✅ **后缀名爆破** - 自动为每个路径追加指定后缀（如 `.php`、`.bak`、`.zip`）
- ✅ **状态码过滤** - 可自定义只显示关心的HTTP状态码
- ✅ **结果排序输出** - 按状态码自动排序，一目了然
- ✅ **结果保存** - 支持将发现的路径导出到文件
- ✅ **智能URL拼接** - 自动处理各种URL格式，防止双斜杠等问题
- ✅ **容错处理** - 自动忽略超时、连接失败等错误，不中断扫描
