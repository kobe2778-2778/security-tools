# SQL注入漏洞扫描器

一款自动化SQL注入漏洞检测工具，支持报错注入检测和时间盲注检测，用于发现Web应用中GET参数存在的SQL注入漏洞。

## ✨ 功能特点

- ✅ **报错注入检测** - 通过分析服务器返回的错误信息判断注入点
- ✅ **时间盲注检测** - 通过响应时间差异判断注入点
- ✅ **多参数自动识别** - 自动提取URL中所有GET参数逐一测试
- ✅ **丰富载荷库** - 内置多种数据库（MySQL、MSSQL、Oracle、PostgreSQL）的测试载荷
- ✅ **实时进度条** - 使用 tqdm 显示测试进度，清晰直观
- ✅ **结果保存** - 支持将发现的漏洞导出到文件
- ✅ **灵活控制** - 可单独启用或禁用某种检测方式
