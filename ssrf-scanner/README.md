# SSRF漏洞扫描器

一款自动化SSRF（服务端请求伪造）漏洞检测工具，检测Web应用是否会根据用户输入发起服务端请求，可能导致内网探测、敏感信息泄露。

## ✨ 功能特点

- ✅ **智能参数识别** - 自动识别URL、IP、域名等疑似SSRF参数
- ✅ **多种检测方式** - 内容回显、时间延迟、重定向、错误回显、连接拒绝
- ✅ **云元数据探测** - 内置AWS、阿里云等云服务器元数据地址
- ✅ **内网服务探测** - 测试常见内网端口（80、22、3306、6379、9200等）
- ✅ **多协议支持** - HTTP、HTTPS、file、dict、gopher
- ✅ **实时进度条** - 使用 tqdm 显示测试进度
- ✅ **结果保存** - 支持将发现的漏洞导出到文件
