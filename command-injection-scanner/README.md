# 命令注入漏洞扫描器

一款自动化命令注入（Command Injection）漏洞检测工具，支持基于响应的特征检测和时间盲注检测。

## ✨ 功能特点

- ✅ **双模式检测** - 基于响应的特征匹配 + 时间盲注检测
- ✅ **智能基线对比** - 先获取正常响应，排除误报
- ✅ **多平台支持** - 可识别Linux、Windows命令执行输出
- ✅ **丰富载荷库** - 覆盖管道符、逻辑运算符、命令替换等多种注入方式
- ✅ **实时进度条** - 使用 tqdm 显示测试进度
- ✅ **结果保存** - 支持将发现的漏洞导出到文件
- ✅ **灵活控制** - 可单独启用或禁用某种检测方式
