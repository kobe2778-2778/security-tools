# 路径遍历漏洞扫描器

一款自动化路径遍历（Path Traversal）漏洞检测工具，检测Web应用是否存在目录穿越/任意文件读取漏洞，可能导致敏感文件泄露。

## ✨ 功能特点

- ✅ **智能参数识别** - 自动识别file、path、page等文件相关参数
- ✅ **多种绕过方式** - 内置URL编码、双编码、空字节截断、双写等绕过载荷
- ✅ **文件内容匹配** - 自动识别/etc/passwd、Windows配置、SSH密钥等文件特征
- ✅ **响应异常检测** - 对比正常响应长度，发现异常的大文件读取
- ✅ **多平台支持** - 同时支持Linux和Windows系统的敏感文件探测
- ✅ **PHP封装器** - 支持php://filter读取PHP源码
- ✅ **实时进度条** - 使用 tqdm 显示测试进度
- ✅ **结果保存** - 支持将发现的漏洞导出到文件
